API reference for Angular SACL badge

import { BadgeModule } from 'sbb-angular';

Directives

BadgeComponent

Selector: sbb-badge

Properties
Name Description
@Input()

active: true

The badge has two status, but, basically its common usage is in active state (see ). Anyway this property can change its status, and so it will change the styling.
@Input()

position: 'default' | 'top'

The position of the badge in a element. The 'default' value is used into component, while the 'top' one is used into component.

badgeStyleClass: boolean

Class activated when the position property is valued with 'top'

cssClass: true
