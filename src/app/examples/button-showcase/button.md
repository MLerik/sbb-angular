# Button Overview

Import button module into your application

```ts
import {
  IconCommonModule,
  ButtonModule
} from 'sbb-angular';
```
and for example you can use the basic primary mode how in the example

```html
  <button sbbButton mode="primary">
      <sbb-icon-arrow-thick *sbbButtonIcon></sbb-icon-arrow-thick>
      Bezeichnung
  </button>
```
