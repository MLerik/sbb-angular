1. Global settings

SBB Angular library sets basic global typography, display and link styles.

  • Use the custom SBB Font stack. SBBWeb Roman SBBWeb Bold  SBBWeb Light SBBWeb Thin SBBWeb Ultralight
  • The html base font is a SBBWeb Roman size 93.75% | 15px with a line height of 1.7
  • Default body background color is #FFFFFF
  • Default link color is #000000, hover/focus color is #C60018

4. Tables

The table is an actual component but you can use the @include table() mixin to style a custom table.

Don't forget to follow a couple of simple rules for the accessibility and styling. In order to have proper styles must be used the thead tbody tags.

In order to have proper accessibility you have to mark table headings with the proper th tag and the scope attribute set to col for column headers and row for row headers.

Don't forget in presence of a custom caption or legend ( *sbbTableCaption directive ) to put proper aria-describedby and aria-labelledby association. Note that the sbb-table component accept an attribute tableLabelledBy="caption_ID" to set proper aria-labelledby attribute on the table.
